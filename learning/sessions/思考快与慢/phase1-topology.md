## Session 1 · 2026-03-02

### 核心命题（因果链）

1. 人类存在两套认知系统 → 系统1（快速、自动、直觉）与系统2（缓慢、费力、理性）共同驱动所有判断与决策
2. 系统1默认主导 → 因为它速度快且几乎不耗能，系统2是"懒惰的"，只在必要时介入
3. 系统1依赖启发式（heuristics）→ 启发式在大多数情况有效，但在特定条件下系统性地产生偏差（bias）
4. 偏差是可预测的 → 因为启发式的错误模式是稳定的，所以认知偏差不是随机噪音，而是规律性错误
5. 人对自身思维过程缺乏元认知 → 我们通常感觉不到系统1在运作，这导致过度自信是默认状态

### 依赖前提

1. 认知资源有限成立，因为大脑是能耗器官，进化压力选择了"够用即可"的策略
2. 两系统可独立激活成立，因为神经科学证据显示快慢两类加工有不同的神经基础
3. 外部世界有稳定规律成立，才使系统1的模式识别有效——在纯随机环境中，直觉不比随机更好
4. 个体无法随意切换系统成立，因为系统2的激活需要主动努力，且容易被认知负荷削弱

### 关键争议

1. 两系统是否真实存在？部分认知科学家认为是隐喻而非神经实体，实际可能是连续谱
2. 偏差可否通过训练消除？Kahneman 悲观；Gigerenzer 认为启发式在生态效度高的环境中是理性的
3. 前景理论的普适性：损失厌恶效应在 WEIRD 样本中稳健，但跨文化复现率存疑

### 盲区清单

- [ ] "系统1=感性，系统2=理性" → 系统1也包含专家直觉，是经验压缩，不等同于情绪化
- [ ] "知道偏差就能避免偏差" → 了解锚定效应后仍然会被锚定，元认知不等于免疫
- [ ] "系统2总是更正确" → 系统2的错误更难发现，因为披着"理性推导"外衣
- [ ] "峰终定律说明体验不重要" → 峰终定律描述的是记忆评价，记忆与体验是两个独立系统
- [ ] "损失厌恶意味着应该冒更多风险" → 描述性理论 ≠ 规范性建议
