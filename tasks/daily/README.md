# 每日日清单

- 每天早上 8:00 自动生成当天文件。
- 文件目录：`tasks/daily/`
- 当天快捷入口：`tasks/daily/today.md`，它是指向当天正式文件的软连接。
- 同一次生成会顺手创建问题日记：`tasks/journal/YYYY-MM-DD.md`
- 问题日记快捷入口：`tasks/journal/today.md`，它也是软连接。
- 生成逻辑：读取昨天文件里所有未完成的 `- [ ]` 项，带入今天的“昨日待办”。
- 每次运行脚本时，会把 `today.md` 重新指向当天日期文件，而不是复制一份内容。
- 今天的模板会按“本周主线 + 今日主内容 + 今日辅助内容 + 今日承接 + 今日复盘”生成，你可以直接改。

## 文件格式

```md
# 2026-03-08 日清单

## 昨日待办
- [ ] 昨天未完成项

## 本周主线
- [ ] 本周最重要的单点突破是什么？

## 今日主内容
- [ ] 题目：
- [ ] 钩子：
- [ ] 最小发布动作：

## 今日辅助内容
- [ ] 题目：
- [ ] 作用：补实验 / 补判断 / 补互动 / 补复盘
- [ ] 最小发布动作：

## 今日承接
- [ ] 评论、私信、主页或成交入口推进 1 次

## 今日复盘
- [ ] 今天最重要的单点突破是什么？
- [ ] 明天要延续什么？
```

## 手动运行

```bash
python3 tasks/scripts/daily_brief.py
```
