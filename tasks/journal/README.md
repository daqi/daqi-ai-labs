# 问题日记

用于记录每天遇到的真实问题，不求完整，只求可复盘。

建议按日期建文件：`YYYY-MM-DD.md`

每天运行 `python3 tasks/scripts/daily_brief.py` 时，会自动创建当天的问题日记，并把快捷入口 `tasks/journal/today.md` 重定向到当天文件。

每条问题尽量写清 4 件事：

- 问题是什么
- 它出现在哪个场景
- 你当时怎么判断或处理
- 后面要不要沉淀成流程、选题、产品或 skill
