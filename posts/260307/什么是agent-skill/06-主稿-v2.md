## 主稿

- 标题：什么是 skill？我用一个真实案例，3 分钟给你讲明白
- 开头：很多人在用 agent，可一提到 skill，外面讲法很多，很多人听了半天还是没感觉。今天我不空讲，直接跑一个真实 skill 给你看。你看完就有感觉了。
- 正文：

  我们一起来看一下，这个 learning-loop 是怎么跑的。

  比如我跟它说，我想学习纳瓦尔宝典。

  你看，我一说完，它就开始自己往下跑了。

  它会先把目录建起来，把脑图文件准备好。

  一本新书，默认就是从第一轮开始。

  第一轮一上来，它会先问我几个问题。

  先判断我对这本书现在是什么水平。

  然后先给我一个知识水平的评分。

  再按照这个水平，给我生成这一轮的课程。

  接下来就是学习、问答、再评分。

  如果还没学会，它就继续往下一轮走。

  而且每走完一步，它都会把结果更新到记忆里。

  所以下次你再来，它就知道你上次学到哪了。

  你看到这儿，大概就明白了。

  它更像 Agent 里的一个小程序。

  有交互，有记忆，跟普通软件没有区别。

- 结尾:

  所以 skill 本质上是拿来运行的。

  你先不用急着记定义。

  你先想一件事，你最想让 Agent 稳定帮你跑的，到底是什么。

  下期我就拿这个问题继续拆，skill 到底是怎么一步步做出来的。
  
  这么长的 SKILL.md，到底是不是手写的。
