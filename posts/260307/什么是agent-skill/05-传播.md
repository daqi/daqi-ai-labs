# 05 传播

## 核心传播判断

- 一句话核心判断：很多人讲了半天你还是没懂 skill，不是因为 skill 太复杂，而是因为没人先把它跑给你看。
- 目标读者：已经在用 agent、准备自己写 skill 的创作者
- 最反常识的点：skill 不是一段更复杂的提示词，它更像一个挂在 agent 上、能被触发和执行的小应用
- 最值得强调的收益或代价：如果你一直把 skill 当提示词，你会写很多说明，但写不出真正能工作的 agent skill

## 标题方向

### 利益角度

- 标题 1：什么是 agent skill？我用一个真实案例，3 分钟给你讲明白
- 标题 2：如果你准备自己写 skill，先把 agent skill 这件事彻底听懂

### 反差角度

- 标题 1：很多人都在讲 skill，但他们讲错了重点
- 标题 2：skill 不是提示词，真正的 skill 更像一个挂在 agent 上的 APP

### 身份或代价角度

- 标题 1：给正在写 skill 的人一句提醒：别把 skill 写成说明书
- 标题 2：如果你已经在用 agent，却还把 skill 当模板，你大概率还没真正懂它

## 开头钩子

- 版本 A：很多人都讲过 skill，但你可能还是不理解。不是你没听懂，是他们一直在解释定义，没有先让你看到它怎么干活。
- 版本 B：我不跟你空讲什么叫 skill，我直接把我最近做的 learning-loop 跑给你看。你看完这个过程，就知道为什么它应该叫 agent skill。

## 金句

- 金句候选 1：skill 就像 Agent 里的微信小程序，你点开它，它不是给你看说明书，它是直接开始干活。
- 金句候选 2：判断一个东西是不是 skill，不看它写了多少文件，而看它能不能被 agent 调起来干活。

## 转发动机

- 为什么别人愿意转发：因为它解决的是很多正在用 agent 的人都卡住的一个真问题。大家都听过 skill，但真要自己写时，脑子里往往还是糊的。这篇给了一个能立刻建立心智模型的解释框架。
- 更适合收藏还是讨论：更适合收藏，也适合转发给“正在折腾 agent 和 skill 的朋友”

## 待用户确认

- 选哪组标题方向：已确认，标题采用“什么是 agent skill？我用一个真实案例，3 分钟给你讲明白”
- 选哪个开头钩子：已确认，版本 B
- 选哪句金句：已确认，大意采用“skill 就像 Agent 里的微信小程序”，候选 2 作为正文里的补刀句
